# Projeto de Baralho Front - End

Projeto feito para o exame de Front-end usando Next.js.

## Como rodar

1. Instalar dependências:
```
npm install
```

2. Criar arquivo `.env.local` na raiz:
```
AUTH_PASSWORD=senha
```

3. Rodar o projeto:
```
npm run dev
```

4. Acessar: 

## Funcionalidades

- Login com senha
- Visualização do baralho completo
- Filtro por naipe
- Embaralhar cartas
- Salva no localStorage
